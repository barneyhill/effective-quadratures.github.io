Regression¶

Finding coefficients via regression.

class equadratures.polyreg.Polyreg(parameters, basis, training_inputs=None, fun=None, training_outputs=None, training_grads=None, quadrature_rule=None, no_of_quad_points=None, gradmethod=None)[source]¶

The class defines a Polyreg object. It is the child of Poly. :param Parameter parameters:

A list of parameters.

Parameters

  • basis (Basis) – A basis selected for the multivariate polynomial.

  • training_inputs (matrix) – A N-by-d matrix of input training points.

  • training_outputs (matrix) – A N-by-1 matrix of output training points.

  • training_grads (matrix) – A N-by-d matrix of gradient evaluations.

  • fun (callable) – Instead of specifying the output training points, the user can also provide a callable function, which will be evaluated.
