class equadratures.induced_distributions.InducedSampling(order, dimension, sample_size, param_type, shape_parameter_A=None, shape_parameter_B=None)[source]¶

This class is used for computing the samples from a polynomial induced distribution

Seshadri, P., Iaccarino, G. and Tiziano, G. Quadrature Strategies for Constructing Polynomial Approximations, Springer 2018

Narayan, A. Computation of Induced Orthogonal Polynomial Distributions, numerical analysis ETNA 2017

Cohen, A. and Migliorati, G. Optimal Weighted Least Square Methods SMAI Journal of Computational Mathematics 2017

Parameters

  • order (int) – type: int Usage: Specify the degree of polynomials used in approximation Example: degree = 3

  • dimension (int) – type: int Usage: Specify the input dimension of the function to be approximated Example: dimension = 6

  • sample_size (int) – type: int Usage: Specify number of samples for the quadrature estimation Example: sample_size = 60

  • param_type (string) – type: String Usage: Specify type of probability measure used Example: “Uniform”, “Beta”

  • shape_parameter_A (string) – type: float Usage: The first shape parameter for a class of probability measures Example: alpha value for freud meaure or beta value for jacobi measure

  • shape_parameter_B (string) – type: float Usage: The second shape parameter for a class of probability measures Example: rho value for freud meaure or alpha value for jacobi measure
