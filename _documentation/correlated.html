Correlation mapping for polynomials¶

Utilities for dealing with correlated inputs.

class equadratures.correlations.Correlations(D=None, R=None)[source]¶

The class defines a Nataf transformation.

The input correlated marginals are mapped from their physical space to a new standard normal space, in which points are uncorrelated. Attributes of the class: :param list D:

List of parameters (distributions), interpreted here as the marginals.

Parameters

  • R (numpy-matrix) – The correlation matrix associated with the joint distribution.

  • std (object) – A standard normal distribution

  • A (numpy-matrix) – The Cholesky decomposition of Fictive matrix R0, associated with the set of normal intermediate correlated distributions.

References for theory:

Melchers, R., E. (Robert E.), 1945- Structural reliability analysis and predictions - 2nd edition - John Wiley & Sons Ltd.

get_correlated_from_uncorrelated(X)[source]¶

Method for mapping uncorrelated variables from standard normal space to a new physical space in which variables are correlated. Input matrix must have [mxN] dimension, where m is the number of input marginals. :param numpy-matrix X:

A Matrix of M-by-N dimensions, in which uncorrelated marginals are organized along rows.

Returns

A N-by-M matrix in which the result of the inverse transformation applied to the i-th marginal is stored along the i-th column of the ouput matrix.
