Nataf utilities¶

Utilities for dealing with correlated inputs.

class equadratures.nataf.Nataf(D=None, R=None)[source]¶

The class defines a Nataf transformation. References for theory:

Melchers, R., E. (Robert E.), 1945- Structural reliability analysis and predictions - 2nd edition - John Wiley & Sons Ltd.

The input correlated marginals are mapped from their physical space to a new standard normal space, in which points are uncorrelated.

Attributes of the class: :param list D:

List of parameters (distributions), interpreted here as the marginals.

Parameters

  • R (numpy-matrix) – The correlation matrix associated with the joint distribution.

  • std (object) – A standard normal distribution

  • A (numpy-matrix) – The Cholesky decomposition of Fictive matrix R0, associated with the set of normal intermediate correlated distributions.

C2U(X)[source]¶

Method for mapping correlated variables to a new standard space. The imput matrix must have [Nxm] dimension, where m is the number of correlated marginals.

Parameters

X (numpy-matrix) – A N-by-M Matrix where input marginals are organized along columns M represents the number of correlated marginals

Returns

A N-by-M Matrix which contains standardized uncorrelated data. The transformation of each i-th input marginal is stored along the i-th column of the output matrix.

CorrelationMatrix(X)[source]¶

The following calculations check the correlation matrix of input arrays and determine the covariance matrix: The input matrix mush have [Nxm] dimensions where m is the number of the marginals.

Parameters

  • X – Matrix of correlated data

  • D – diagonal matrix which cointains the variances

  • S – covariance matrix

Returns

A correlation matrix R

U2C(X)[source]¶

Method for mapping uncorrelated variables from standard normal space to a new physical space in which variables are correlated. Input matrix must have [mxN] dimension, where m is the number of input marginals.

Parameters

X (numpy-matrix) – A Matrix of M-by-N dimensions, in which uncorrelated marginals are organized along rows.

Returns

A N-by-M matrix in which the result of the inverse transformation applied to the i-th marginal is stored along the i-th column of the ouput matrix.
